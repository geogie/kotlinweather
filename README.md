# kotlin weather

说明：  
1. Gson、retrofit、rxJava+RxAndroid
2. 简单的天气练习项目：横竖屏适配(layout-land)、FragmentDialog等
3. 手机适配（hdpi、xdpi、xxhdpi等）[理解](http://blog.csdn.net/lincyang/article/details/44174997)
